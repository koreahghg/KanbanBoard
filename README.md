# Kanban Board

Trello 클론 프로젝트. Next.js App Router 기반 칸반 보드.

## 기술 스택

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **State**: Zustand
- **Styling**: Tailwind CSS v4
- **DnD**: dnd-kit

## 시작하기

```bash
npm install
npm run dev
```

## 스크립트

| 명령어 | 설명 |
|--------|------|
| `npm run dev` | 개발 서버 실행 |
| `npm run build` | 프로덕션 빌드 |
| `npm run lint` | ESLint 검사 |
| `npm run format` | Prettier 포맷팅 |
